Inigo

Installation Package
Source Code

Inigo is a SQL 2008 package creation interface which consists of the following applications:

  1. Inigo Builder
  2. Inigo Deployment

Excerpt from Inigo Builder:

The Intro

The Inigo Builder utility was created to simplify database scripting and deployment to remote sites. Often when deploying database patches, or complete builds, there are a variety of concerns:

  1. Sharing of database credentials.
  2. Database scripts stored in clear text format.
  3. Version control.

The Mission

The goal was to bring a combination of separate projects and utilities in to a collaborative project meeting these needs. The utility provides the ability to do the following:

  1. Create full build scripts of each installation.
  2. Create patches with the ability to move up or down in revision.
  3. Secure database information for remote deployments.
  4. Handle version updates.
  5. Encrypt deliverables with FIPS compliant algorithms.
  6. (Future) Push deliverables to remote servers.

The Setup

Initially the database administrator will need two databases. The "Source" database being the development server with the recent changes. The "Target" database will be the fielded or working server to which the changes should be applied.

Creation of the full build script will go solely from the "Source" server since the patch will include the complete build script on each deployment (preferred).
