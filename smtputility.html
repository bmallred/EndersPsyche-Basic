Smtp Test Utility

Executable
Source Code

This is a simple utility used to test Smtp message properties and sending. It supports all the basic mail message properties for a quick release, but it also supports several more advanced features. In a business environment I found some custom Smtp servers do not behave exactly as expected so testing of various encodings were needed as well as the ability to different authentication methods as well.

Authentication

  • User name, password, and/or domain capabilities.
  • Secure Socket Layer (SSL) compliance.

Advanced

  • Ability to switch subject and body encoding on the fly.
  • Different tracing levels for diagnostic needs.
