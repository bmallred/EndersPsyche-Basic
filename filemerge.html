File Merge

Source Code

There a couple of reasons why this was developed.

  1. Our SQL Developer has a program to store all database objects in single files. This allows for easier subversion control on an itemized level. From there a full build script would be required on each new build but the individual files were not used because it would take just as much time as piecing it together by hand.
  2. One project I work on is an application suite with multiple projects tied together but still separate. Each of these required their own change logs, but there was a need to create one compiled version for the suite.

Thus spawned the file merge application. I know most of these problems can be solved via command-line manipulation, but we also wanted to give ourselves the ability to format file contents using templates (simple String.Format() works for us).

The user can select either an entire directory or specific files. If a directory is selected all subdirectories and associated files will be added to the merge file (word of caution!). I did not include an easy template creation process and for now will have to add them manually edit these in the settings XML file.
