Visual Trace Route

Source Code

A collaboration between Ed Rantanen and myself to provide basic visualization of a trace route in a flexible manner. This should allow the swapping of different graphic libraries such as Graphviz, Flot, etc. Due to the inherit flexibility the key is all in how the data structure is formed. Taking lessons from The Pragmatic Programmer (see the library!) we use some basic plaintext to parse and create data structures on the fly.

Examples of graphing libraries currently includes:

  1. Flot
