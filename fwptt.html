Fast Web Performance Test Tool

Source Code

From the original website:

fwptt [is] a Web application tester program for load testing web applications. It can record normal and AJAX requests. I tested it on ASP.Net applications, but it should work with JSP, PHP or other[s].

The beginning

Essentially it does a couple of tasks:

  • Records browser sessions
  • Allows the editting of these sessions and their attributes
  • Plays the recorded session with returned statistics

Reason for a new beginning

The reason for the re-vamp of the program was due to a couple of issues:

  • Recording did not properly record HTTPS sessions
  • Two many options made navigation difficult within the application
  • Translated an XML file to C# code which then had to be compiled before execution of "Playback" functionality (i.e. too many steps involved to return the end results)
  • Threading crashed the application
  • Non-technical individuals required time to learn the tool

I personally enjoyed the program despite these "hills" because I consider myself to be a fairly proficient and knowledgeable techno-geek. However, I needed my peers to be able to quickly run tests against our software with no training required. Due to enjoying the techniques and program itself I adopted many of it's features and aimed to improve upon them as much as possible. Some of the major changes which have been made are the following:

  • Strictly using XML in all file formats
  • Simplified the process from six (and potentially more) steps to three
  • Re-worked datahandling processes
  • Fixed HTTPS issue(s)
  • Fixed Threading and implemented C# BackgroundWorker model
  • Added use of X509 certificates
  • Report creation

The outcome

After the revisions were complete I gave the product back to the team for their use and within five minutes (no exaggeration) or less they were compiling recordings for our unit, integration, and load tests.
